=== Ghost Importer ===
Tags: importer, blog, cms

Import posts, images, tags and settings from Ghost into Wordpress.

== Description ==
Generate an export file from your Ghost blog admin and use this plugin to import posts, their tags and a few translatable settings to your Wordpress installation. Images can also be copied across if you provide the URL for your Ghost blog.
